Doraibu (ドライブ) is a Japanese culture focused Discord bot.

## Commands

### d!help

Obvious.

### d!mal <anime name>

Fetch info from an anime. (MAL)

### d!mml <manga name>

Fetch info from a manga. (MAL)

### d!osu <username>

Fetch osu! user info.

### d!ping

Ping-pong.
